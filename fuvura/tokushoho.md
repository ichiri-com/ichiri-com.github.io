## サービス情報

| 項目 | 内容 |
| :--- | :--- |
| 販売URL | App Store（アプリ「Fuvura」） |
| 販売価格 | 各サブスクリプション商品のApp Store販売ページに表示される価格（消費税込）に従います。本書面公開時点の価格：プレミアムプラン 月額 ¥480（税込） |
| 商品代金以外の必要料金 | 消費税（販売価格に含む）。インターネット接続に必要な通信料金は利用者負担。 |
| 支払方法 | Apple ID に登録された決済手段（クレジットカード、キャリア決済、Apple ID 残高 等） |
| 支払時期 | 購入時および各更新日に Apple ID アカウントへ請求 |
| サービス提供時期 | 決済完了後、直ちに有料機能を利用可能 |
| 自動更新 | サブスクリプションは、現在の課金期間の終了日の24時間以上前に解約手続きを行わない限り、自動的に同一条件で更新されます。 |
| 解約方法 | iOS 端末の「設定」>「Apple ID」>「サブスクリプション」よりお客様ご自身で手続きいただけます。本アプリのアンインストールではサブスクリプションは解約されません。 |
| 返金 | 課金期間の途中で解約された場合でも、未経過期間に対する料金の返金は原則として行いません。返金については、Apple Inc. の返金ポリシーに従います。 |
| 動作環境 | iOS / iPadOS 17.0 以上を搭載した iPhone / iPad、および macOS 14.0 以上を搭載した Apple Silicon Mac（Designed for iPad） |
| 特別な販売条件 | 本アプリは投資助言、投資勧誘または金融商品の取引を行うものではありません。シミュレーション結果は将来の運用成果を保証するものではありません。 |
